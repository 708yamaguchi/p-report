# p-report
Latex template for short thesis
---

### 1. Prerequisities

```bash
$ sudo apt-add-repository ppa:texlive-backports/ppa
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install texlive-lang-cjk
```

### 2. Edit `p-report.tex`

### 3. Make pdf

```bash
$ make
```

### Optional. cleaning

```bash
$ make clean
# or
$ make wipe
```
